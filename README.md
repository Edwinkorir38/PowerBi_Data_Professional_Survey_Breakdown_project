# Data Professional Survey Breakdown – Power BI Project

##  Overview
This project analyzes a dataset of **630 data professionals** to uncover insights about demographics, salaries, tools, job satisfaction, and the difficulty of entering the data industry.  
The analysis and dashboard were fully developed using **Power BI Desktop**.

The goal of this project is to demonstrate practical skills in **data cleaning, modeling, visualization, and data storytelling**, while producing an interactive dashboard that highlights key industry trends.

---

##  Tools Used
- **Power BI Desktop** – visualization & dashboard creation  
- **Power Query** – data cleaning & transformation  
- **DAX** – custom calculations and measures  

---

##  Dataset
The dataset includes responses from **630 survey participants** working across various data roles, including:
- Data Scientist  
- Data Analyst  
- Data Engineer  
- Database Developer  
- And more…

Additional variables include:
- Country of residence  
- Salary  
- Favorite programming language  
- Happiness scores  
- Difficulty entering the field  
- Demographics  

---

##  Dashboard Features

### **1. Country of Survey Takers**
Treemap showing the distribution of respondents across countries (Canada, India, UK, etc.).

### **2. Average Salary by Job Title**
Bar chart comparing average salaries across data roles, with Data Scientists and Data Architects earning the highest averages.

### **3. Favorite Programming Language**
Bar chart revealing that **Python** is the most popular programming language among respondents.

### **4. Difficulty Breaking Into Data**
Donut chart illustrating how challenging participants found entering the data profession.

### **5. Happiness Metrics**
Two gauges displaying:
- **Average work-life balance satisfaction**
- **Average salary satisfaction**

### **6. Summary Statistics**
- **630** completed survey responses  
- **29.87** average age of participants  

---

##  Key Insights
- Python is the top programming language among data professionals.  
- Salary varies significantly by job title.  
- Many respondents reported moderate difficulty breaking into data.  
- Work-life balance satisfaction is higher than salary satisfaction.
